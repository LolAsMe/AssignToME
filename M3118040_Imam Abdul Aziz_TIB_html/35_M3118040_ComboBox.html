1. Combo Box

Seperti halnya radio button, combo box juga digunakan untuk input yang berupa pilihan. Penampilan combo box mirip seperti radio button namun terdapat drop down button untuk menampilkan item pilihan. Untuk membuat combo box, tag awalnya adalah <select name="namakomponen"> dan diakhiri dengan </select>. Sedangkan untuk item pilihannya diapit dengan tag &tl;option value="nilai_item"> dan </opteion>.
