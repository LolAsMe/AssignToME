Dasar HTML5 dan Penerapannya

1. Membuat Link

Fungsi dari link adalah menghubungkan satu halaman web ke halaman lain (baik dalam situs yang sama maupun tidak). Cara mengaktifkan link hanya dengan mengklik suatu objek( dapat berupa teks maupun gambar, bisa juga dalam bentuk komponen form). Untuk membuat link dalam bentuk teks atau gambar caranya dengan memberi perintah < href="url"> dan diakhiri dengan </a>. Kepanjangan dari href adalah hypertext reference. Dalam hal ini url dapat diisi dangan halaman web atau situs web yang ingin dilink-kan.

2. Script 

<h1> Link </h1>
<p>
Klik link berikut <a href="page2.html"> Link
</p>

3. Output

Link

Klik link berikut Link

4. Kesimpulan

Link digunakan untuk memudahkan pengunjung untuk menuju ke halaman web yang bersangkutan (menuju sub-halaman) dengan mudah sehinngga terjadai efisiensi dalam pengggunaan. Dan juga apabila kursor diletak di link tersebut kita dapat melihat cuplikann dari nama halaman yang dituju.
