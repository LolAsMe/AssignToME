Dasar HTML5 dan Penerapannya

1. Unordered List

Dalam HTML terdapat tag untuk membuat list, baik berupa unordered list maupun ordered list. Maksud ordered list yaitu list yang setiap itemnya terdapat huruf atau angka yang terurut. Sedangkan unordered list merupakan list yang tidak terurut, biasanya berupa bulleted list ( list dengan simbol tertentu ). Untuk membuat unordered list(ul), caranya dengan menuliskan <ul> dan diakhiri dengan </ul>. Sedangkan daftar item ditulis diantara tag <ul> dan </u> tersebut, dengan diapit dengan tag <li> dan </li>

2. Script 

<li>Jeruk</li>
<li>Apel</li>
<li>Duku</li>
<li>Stroberi</li>
<li>Nanas</li>

3. Output

  • Jeruk
  • Apel
  • Duku
  • Stroberi
  • Nanas

    • 4. Kesimpulan

    HTML memberikan fitur untuk membuat list untuk memudahkan membuat list secara mudah tanpa harus memikirkan format dan posisi. Hal ini sangat berguna dalam membuat web.
