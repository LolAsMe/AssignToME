1. Favorite Icon(Favicon)

Favicon (Favorites Icon) adalah suatu image(icon) yang akan tampil pada address bar, menu favorite, dan lain lain pada browser. Favicon digunakan untuk menandai web sehingga mudah dikenali sehingga tidak perlu repot untuk melihat satu persatu di favorite bar. Untuk membuat favicon, terlebih dahulu Anda harus memiliki image berupa icon berukuran 16 x 16 pixel, usahakan dalam bentuk .ico .Kita juga bisaa membuat sendiri dengan mudah file extension icon yaitu dengan menggunakan web contoh nya favicon.cc dan melalukan langkah langkah yang sudah disediakan. penempatan perintah membuat favicon terletak ddidalam block header.
