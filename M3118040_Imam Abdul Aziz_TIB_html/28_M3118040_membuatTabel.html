Dasar HTML5 dan Penerapannya

1. Pendahuluan

Tabel merupakan struktur yang digunakan untuk menampilkan informasi dalam bentuk baris dan kolom. Pada HTML, untuk pembuatan tabel menggunakan <table> untuk membuat tabel, <tr> untuk membuat baris dan <td> untuk membuatkolom Untuk lebih memahami pembuatan tabel pada HTML Maka diperlukan tag-tag. Tag-tag tersebut ialah :
  1. <table> merupakantag pembuka dalam membuat sebuah tabel pada html, tanpa <tabel> ini, penggunaan <tr> dan <td< tidak bisa berfungsi
  2. <tr> atau dikenal dengan tabel row merupakan tag yang digunakan untuk membuat baris dalam tabel HTML.
  3. <td> atau dikenal dengan tabel data merupakan tag yang digunakan untuk membuat kolom dalam baris dan untuk menampilkan data dalam tabel.
  4. <th> atau dikenal dengan tabel Head atau bagian kepala pada tabel.

2. Script 

<h3>Harga Buah</h3>
<table>
<tr>
<th>Nama Buah</th>
<th>Harga/Kg</th>
</tr>
<tr>
<td>Apel</td><td>Rp. 8.000,- </td>
</tr>
<td>Jeruk</td><td>Rp. 7.500,- </td>
<tr>
<td>Klengkeng</td><td>Rp. 10.000,- </td>
</tr>
<tr>
<td>Anggur</td><td>Rp. 12.000,- </td>
</tr>
		</table>

4. Kesimpulan

Tag yang paling penting dalam pembuatan tabel dalam HTML ialah tag <table>, <tr>, <td>. Sementara tag yang lain adalah tambahan ( opsional ), boleh digunakan boleh tidak
