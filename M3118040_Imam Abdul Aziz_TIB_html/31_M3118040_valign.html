Dasar HTML5 dan Penerapannya

1. Pendahuluan

Attribut Valign dan Align dalam memformat sel digunakan untuk mengatur posisi data pada cell. Valign digunakan untuk mengatur posisi vertikal data relatif terhadap sel, sedangkan align mengatur posisi horizontal. Secara default nilai atau value dari Valign pada cell adalah center dan Alignnya adalah left.

2. Script

<h1 align="center"> Harga Buah </h1> <table border="2" align="center" width="40%"> <tr> <td rowspan="3" valign="top"> Ini adalah Tabel</td> <td> Hypertext Markup Languange</td> </tr> <tr> <td align="right">HTML</td> </tr> <tr> <td> Perintah dan Tahapan Web Dasar</td> </tr> </table>

3. Output

Harga Buah

Ini adalah Tabel Hypertext Markup Languange
HTML
Perintah dan Tahapan Web Dasar

4. Kesimpulan

Dari percobaan diatas dapat kita simpulkan bahwa, Pada cell "ini adalah tabel" posisi teks dalam cell berada di atas (Berdasarkan lokasi vertikal), dan di sel "HTML" posisi teks dalam cell berada di bagian kanan(Berdasarkan Posisi Horizontal)
