Dasar HTML5 dan Penerapannya

1. Form Login

Komponen text box dibuat dengan menuliskan tag <input type="text" name="nama komponen">. dengan attribute name diisidengan nama yang sesuari dengan data isian. Nama komponen mutlak dituliskan karena akan digunakan pada script pengolah data. Untuk text box password ada tag khusus untuk itu, karena digunakan untuk memasukkan data rahasia. Tag tersebut ialah <input type="password" name="password">

2. Script

<form method="post" action="form.html">
Nama :
<br/>
<input type="text" name="firstname" value="Nama">
<br />
Password :
<br/>
<input type="password" name="passwordku" value="Password">
<br/>
<br/>
<input type = "submit" value="Submit">
</form>
<p>
Jika anda submit data melalui tombol submit maka data akan ditampilkan pada halaman "form.html"
</p>
