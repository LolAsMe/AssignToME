1. Check Box

Seperti halnya radio b utton , check box juga digunakan untuk input berupa pilihan. Penampilanyya pun hampir sama dengan radio button, namun perbedaanya adalah bahwa dalam check box memungkinkan user memilih item lebih dari satu. Tag untuk membuat check box hampir sama dengan radio button seperti dibawah ini.
